The classes in this package pose the core of the Janino JavaTM compiler.

The package comprises a scanner ({@link edu.columbia.psl.invivoexpreval.Scanner}, a parser ({@link edu.columbia.psl.invivoexpreval.Parser}) and a class file library. The parser builds a syntax tree from the "Java.*" classes that represents the parsed code. The {@link edu.columbia.psl.invivoexpreval.UnitCompiler#compileUnit} method compiles this syntax tree into a {@link edu.columbia.psl.invivoexpreval.util.ClassFile} object, which can write JavaTM bytecode to an "OutputStream".
